## Configurar o WhatsApp

No final do arquivo `index.html`, localize:

```javascript
const whatsappNumber = "";
```

Adicione o número oficial com DDI e DDD, somente números:

```javascript
const whatsappNumber = "5575999999999";
```

## Confirmar antes de publicar

- Nome profissional oficial
- Cidade e área de atendimento
- Número de WhatsApp
- Serviços realmente oferecidos
- Pacotes e preços
- Prazos de entrega
- Quantidade de revisões
- Direitos de uso dos vídeos e imagens
- Logo oficial
- Fotografias e trabalhos autorizados
